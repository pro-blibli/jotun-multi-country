Build Flexible Campaign Rules

Create campaign logic that rewards your stakeholders based on product categories and purchase behavior

🎯

Configure Campaign Rules in Talon.One

Talon.One's Rule Builder lets you create sophisticated campaign logic with conditions (what triggers the campaign) and effects (what rewards are given). Here's how you structure rules for Jotun's paint loyalty program.

Example Campaign Rules for Jotun Paint Scanning

Rule 1

Premium Paint Bonus

ACTIVE

CONDITION

✓ Scanned_Item = true
✓ Product.Category = "Premium Paints"
✓ Session Total ≥ 60,000 VND

EFFECT

⚡ Award Painter Pts = Item.Price × 8%

Rule 2

Architect Rewards

ACTIVE

CONDITION

✓ Scanned_Item = true
✓ Architect ID exists in session
✓ Session Total ≥ 2,000 THB

EFFECT

⚡ Award Architect Pts = Item.Price × 5%

Rule 3

Dealer Commission

ACTIVE

CONDITION

✓ Scanned_Item = true
✓ Dealer ID exists in session
✓ Session Total ≥ 850 MYR

EFFECT

⚡ Award Dealer Pts = Item.Price × 3%

What These Rules Do:

  • Rule 1: Rewards painters when they scan premium paint products above minimum value
  • Rule 2: Rewards architects with points based on scanned product value
  • Rule 3: Rewards dealers with commission points for each transaction
  • All three rules can fire from a single QR code scan—rewarding multiple stakeholders simultaneously
  • Placeholders (highlighted fields) allow market-specific customization without rebuilding logic
🎨

Why Placeholders Matter for Multi-Market Campaigns

Notice the highlighted fields like [painter_pts_percentage] and [min_value_scanned_item]? These are placeholders. Malaysia might offer 10% painter rewards, while Vietnam offers 12% due to competitive dynamics. The same campaign structure adapts to each market's needs—no rebuilding required.

⚡ From Rules to Templates in Minutes

Once your rules are configured with placeholders for flexible values (reward percentages, minimum thresholds), you can save them as a template and deploy across all markets—Malaysia, Vietnam, Philippines, and beyond—without rebuilding the logic each time.

2

Create Campaign from Template

Two flexible ways to deploy campaigns across your markets

🖥️

Option 1: Deploy from Dashboard

Use Talon.One's dashboard interface to manually deploy templates to each market. This gives you full visibility and control at each step of the process.

How It Works:

  • Step 1: Navigate to your target market application (e.g., Malaysia)
  • Step 2: Click "Create Campaign" → "From Template"
  • Step 3: Select your saved template (e.g., "Q2 Paint Scan Rewards")
  • Step 4: Fill in placeholder values specific to that market
  • Step 5: Review and activate the campaign
⏱️

Time Required

Approximately 2-3 minutes per market. For 20+ countries, this method takes around 40-60 minutes total.

🤖

Option 2: Deploy with Claude AI

Use Claude (AI assistant) to create campaigns from your templates using simple, natural language commands. Just describe what you want, and Claude handles the API calls to Talon.One across multiple markets simultaneously.

How It Works:

  • You say: "Deploy the Q2 Paint Rewards template to Malaysia, Vietnam, and Philippines with 10%, 12%, and 11% painter rewards respectively"
  • Claude does: Creates campaigns in all three markets simultaneously with your specified values
  • Result: Multiple campaigns deployed in seconds, not minutes
  • No need to remember API endpoints, navigate UIs, or repeat the same steps 20+ times
💬

Example Command

"Create campaigns from the Paint Scan Rewards template for Malaysia, Vietnam, and Philippines. Use 10% for painters in Malaysia, 12% in Vietnam, and 11% in Philippines. Set minimum purchase to 700 for all markets."

Benefits of Natural Language Deployment:

  • No technical knowledge required—describe what you want in plain English
  • Deploy to multiple markets with one command
  • Reduces deployment time from 40-60 minutes to under 3 minutes
  • Fewer errors—Claude validates parameters before creating campaigns
  • Batch operations: update, extend, or modify multiple campaigns at once
⚡

Time Required

Under 3 minutes for 20+ countries. Claude deploys all campaigns in parallel with a single natural language command.

💡 Which Option Should You Choose?

Dashboard deployment is perfect when you need full visibility and manual control for each market, or when deploying to just a few countries. Claude AI deployment is ideal for rapid multi-country rollouts, recurring campaigns, or when speed and consistency are priorities. Many teams use the dashboard for initial setup and testing, then switch to Claude for production rollouts.
